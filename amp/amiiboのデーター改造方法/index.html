チュートリアルの目的

最近、3DSやWiiUなどの改造が話題になっていますが、

Amiiboまで改造することができるようになりました。

あまり利用する機会は無いかと思いますが、ここにまとめておきます。

必要な物

  • NFC対応のAndroidデバイス

  • TagMo2.1（詳しくは下のリンクの動画の説明欄から）

    https://www.youtube.com/watch?v=FzSyBjZM5XE

  • unsafe Amiiwrite

    http://apkshared.net/1jav

  • Hex Editor Free （Playストアで検索してインストールして下さい。）

前提条件

  • 動画「Amiiboのデーターをコピーしよう！」のSTEP1の段階を終えていること。
    https://www.youtube.com/watch?v=DPR1UPhLkbw
  • Hex Editor Free、ESファイルエクスプローラーをインストールしていること。

    • やり方

    1. 「TagMo」でAmiiboのbinをダンプ
    2. 「Hex Editor Free」でbinファイルの末尾にバイナリエディタで00を540バイトになるまで入れる。
    3. 「Amiibo Cheat Tool by VGMoz」で改造し、改造データをダウンロード。
    4. 「unsafe Amiiwrite」でAmiiboに書き込み。

    ★詳しい解説は動画をご覧下さい。

    困ったときのFAQ

    Q.HexEditerFreeのファイル選択画面でダンプしたbinファイルが見つかりません。

    A.上の方に「/」が出ている場合は端末のルートディレクトリを参照しています。
    私の場合はルートの「sdcard」ディレクトリから「Download」ディレクトリを発見出来ました。
    ルートの「mnt」ディレクトリ内に「sdcard」がある場合もあるそうです。
    端末によって様々なので「＜端末名＞　フォルダ構成」等で検索し見つけて下さい。

    Q.TagMoの画面が動画内のと違うのですが…

    A.TagMoのAPKをTagMo2.1に差し替えてみて下さい。
    https://drive.google.com/file/d/0BwV_DUnv_6kxVkJLRFpfaU9aNTQ/view?usp=sharing

    Q.NFCタグを端末にかざしても何も起きません。

    A.まずは、その端末が本当にNFC対応か確認して下さい。「＜端末名＞ NFC」などで検索してみて下さい。
    対応している場合は、設定アプリを開き、無線とネットワークの「その他の設定」や「もっと見る」を選択してください。
    その画面からNFCを利用可能に設定できると思います。
    （端末によって名称がまちまちなので実際に行って確認してみてください。）

    Q.NTAG215クローンしたAmiiboでも利用できますか。

    A.はい。クローンしたAmiiboは普通のAmiiboと同じように振る舞いますので、利用出来ます。

    Q.APKをインストールしようとすると「インストールはブロックされました」などと出てインストールできません。

    A.設定アプリを開きセキュリティから「提供元不明のアプリ」のインストールを許可してください。
    （端末によって名称や設定方法が違うことがあります。）

    Q.改造に対応できるゲームを教えて下さい。

    A.「Amiibo Cheat Tool by VGMoz」で改造できるゲームは、
    ・大乱闘スマッシュブラザーズ for Nintendo 3DS / Wii U
    ・マリオパーティ10
    ・マリオテニス ウルトラスマッシュ
    ・ショベルナイト
    ・なげなわアクション! ぐるぐる!ちびロボ!
    です。（2016年6月12日現在）
    また、動画内では紹介していませんが、「Amiibo Hex Editing Tool by VGMoz」を利用すると、対応していないゲームでも自分でバイナリをいじって改造出来るかもしれません。
